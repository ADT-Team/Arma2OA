Различные настройки
- С помощью ножа можно собирать коноплю с плантаций,надо быть рядом с кустом, открыть инвентарь и нажать ПКМ по ножу "Собрать Урожай".
- В палатке можно спать для восстановления здоровья.
- Новая система строительства
- Для любителей острых ощущений можно принимать адреналин самому.
- ПКМ по радио открывает управление группами.
- В трейд-зонах можно бесплатно подлечиться при взаимодействии с "Медик" (Скролл-меню)
КРАФТ
Все действия выполняются, нажимая ПКМ на предмет в инвентаре.
- Canvas(нужен для создания палаток,брезентов,козырьков) = Civilian Clothing + Инструменты(нож и ремкомплект).
- Переплавка Золотого Калаша(20голды) = Золотой Калаш + Огонь в 3-х метрах + Инструменты(ломик и ремкомплект) + Консервная банка.
- Переплавка Золотого Револьвера(5голды) = Золотой Револьвер + Огонь в 3-х метрах + Инструменты(ломик и ремкомплект) + Консервная банка.
- Косяк = Конопля + Документы или газета + Спички.
- 1 голд = 2 золотой руды + фляжка с водой + Огонь в 3-х метрах + Инструменты(кувалда и ремкомплект) + Консервная банка.
- 10oz Сильвер = 5 серебренной руды + фляжка с водой + Огонь в 3-х метрах + Инструменты(кувалда и ремкомплект) + Консервная банка.
- Запчасти = 3 железной руды + фляжка с водой + Огонь в 3-х метрах + Инструменты(кувалда и ремкомплект) + Консервная банка
- Шлакоблок = Workbench(верстак) + Инструменты(ремкомплект и саперная лопатка) + Мешок с песком + Ведро с цементом.
